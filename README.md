# Python Blackjack 🃏

A robust, console-based Blackjack game implemented in Python. 

## Features
- **Smart Ace Logic**: Automatically switches Ace value between 11 and 1 to prevent busting.
- **Robust Input Handling**: Custom validation loop ensures the game doesn't crash on typos.
- **Modular Design**: Clean separation of concerns with dedicated functions for card dealing, scoring, and game logic.
- **Global Configuration**: Easily adjustable rules via constants (Blackjack limit, Dealer threshold, etc.).

## How to Play
1. Run `python main.py`.
2. Follow the on-screen prompts to 'Hit' (get another card) or 'Pass'.
3. Try to get closer to 21 than the computer without going over!

## Requirements
- Python 3.x
